# Corrective RAG & Self RAG

This repository contains implementations and experiments for **Corrective RAG (CRAG)** and **Self RAG** using Jupyter notebooks.  
The project explores retrieval refinement, query rewriting, evaluator-based retrieval, and web-enhanced RAG pipelines.
